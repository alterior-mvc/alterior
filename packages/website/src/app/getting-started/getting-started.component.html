The Alterior CLI provides scaffolding and build tooling to make the development experience seamless, but you can also develop Alterior apps from scratch using the tools you are already familiar with.

Install Alterior CLI in your development environment:

npm install @alterior/cli -g

Create your new project:

alt new myapp

Open the new project in your favorite IDE:

code myapp
